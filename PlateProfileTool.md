# 銅盤剖面圖計算工具說明

## 用途

此工具用來輸入銅盤外徑半徑 `R`、內孔半徑 `r`、量測值 `OD`、`ID`，並畫出銅盤側視剖面圖、車刀弧面路徑，以及 OD / ID / R / r 的平面量測示意圖。

## 基本定義

- `R`：外徑半徑。
- `r`：內孔半徑。
- `A`：外緣位置，Z 基準點。
- `C`：內孔起刀邊位置。
- `X`：A-C 中點位置。
- `OD`：外緣 A-B 量測得到的值。
- `ID`：內孔邊 C 到外緣 A 量測得到的值。

## 盤面 Z 值公式

```text
Za = 0
Zc = OD
Zx = (Za + Zc) / 2 + ID
X 位置 = (R + r) / 2
AC = R - r
```

判斷方式：

```text
ID > 0：凸面，X 高於 A-C 直線
ID < 0：凹面，X 低於 A-C 直線
ID = 0：接近斜平面，X 落在 A-C 直線上
```

## 車刀路徑定義

車刀路徑使用三點弧面：

```text
Z1：C 位置
Z2：AC 中點
Z3：A 位置
```

車刀 Z 軸方向和量表方向相反：

```text
車刀 Z 正值 = 往下
車刀 Z 負值 = 往上
```

工具內部顯示時，以「車刀 Z 零點位置」為基準：

```text
刀具在 C 的圖面高度 = 車刀Z零點位置 - Z1
刀具在 X 的圖面高度 = 車刀Z零點位置 - Z2
刀具在 A 的圖面高度 = 車刀Z零點位置 - Z3
```

## 計算車刀 Z 值

小視窗輸入：

- 實際值：目前車完後量到的 `OD`、`ID`
- 前值：目前使用的 `Z1`、`Z2`、`Z3`
- 目標值：想要達到的 `OD`、`ID`

計算邏輯：

```text
實際 Zc = 實際 OD
實際 Zx = 實際 OD / 2 + 實際 ID
實際 Za = 0

目標 Zc = 目標 OD
目標 Zx = 目標 OD / 2 + 目標 ID
目標 Za = 0

C 修正量 = 實際 Zc - 目標 Zc
X 修正量 = 實際 Zx - 目標 Zx
A 修正量 = 實際 Za - 目標 Za

新 Z1 = 前 Z1 + C 修正量
新 Z2 = 前 Z2 + X 修正量
新 Z3 = 前 Z3 + A 修正量
```

## 平面說明圖標示

- `R`：外圓半徑。
- `r`：內孔半徑。
- `a`、`b`：OD 量測用外緣兩點。
- `c`：ID 量測起點，位於內圓外側。
- `X`：A-C 線段中點。
- `Y`：OD 線段中心點。
- `OD`：A-B 外緣量測線。
- `ID`：C-A 內孔到外緣量測線。
